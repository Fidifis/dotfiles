# dotfiles

## Run

```
curl -sSL https://raw.githubusercontent.com/Fidifis/dotfiles/main/appinstall.bash | bash
```

install as private

```
curl -sSL https://raw.githubusercontent.com/Fidifis/dotfiles/main/appinstall.bash | bash -s -- --private
```
